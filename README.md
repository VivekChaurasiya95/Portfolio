# Vivek Chaurasiya | Personal Portfolio

A sleek, responsive, and highly interactive personal portfolio showcasing professional experiences, technical projects, certifications, and skills. Built with modern web technologies and featuring smooth 3D animations and glassmorphic UI designs.

## 🚀 Features

- **Dynamic Interactive UI:** Built with Framer Motion, featuring 3D tilt effects, scrolling timelines, and beautiful micro-animations.
- **Glassmorphic Design:** A modern dark-space aesthetic integrating Tailwind CSS and semi-transparent glass cards.
- **Interactive Certifications:** A responsive image grid featuring full-screen detailed modals.
- **Journey Timeline:** A chronological animated timeline detailing education and professional experience.
- **Contact Form:** Integrated seamless Email capability for visitors to connect dynamically.

## 🛠️ Technologies Used

- **Framework:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **UI Components:** shadcn/ui & Lucide React
- **Icons:** Lucide Icons

## ⚙️ Getting Started

To run this project locally, follow these steps:

### Prerequisites
- Node.js & npm installed on your machine.

### Installation

1. **Clone the repository**
   ```sh
   git clone <YOUR_GIT_URL>
   cd <YOUR_PROJECT_NAME>
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Set up Environment Variables (Optional)**
   If you want to use the Web3Forms background contact API, create a `.env` file in the root folder and add:
   ```env
   VITE_WEB3FORMS_KEY=your_access_key
   ```

4. **Start the development server**
   ```sh
   npm run dev
   ```
   Open `http://localhost:5173` (or the provided port) in your browser.

## 📄 Customization
All source code is fully accessible within the `/src` directory. To update text content, links, or sections, simply modify the respective component files inside `/src/components/` and the `/src/data/` folders.
