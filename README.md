## ✨ Features

- ✨ **Dynamic Interactive UI** - Smooth 3D tilt effects, scrolling timelines, and beautiful micro-animations using Framer Motion.  
- 🔮 **Glassmorphic Design** - Atmospheric deep-space aesthetics with semi-transparent glass cards tailored using Tailwind CSS.  
- 📜 **Interactive Certifications** - A responsive image grid featuring full-screen detailed modals.  
- ⏳ **Journey Timeline** - A chronological, animated timeline detailing education and professional experience.  
- 📬 **Contact Form** - Seamless Email capability for visitors to connect dynamically.

## ⚙️ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) & `npm` installed.

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/VivekChaurasiya95/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up Environment Variables (Optional)**  
   If you want to use the Web3Forms background contact API, create a `.env` file in the root folder and add:
   ```env
   VITE_WEB3FORMS_KEY=your_access_key
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```
   Open `http://localhost:5173` (or the provided port) in your browser

## 🎨 Customization

All source code is fully accessible within the `/src` directory. To update text content, links, or sections, simply modify the respective component files inside `/src/components/` and the `/src/data/` folders.
